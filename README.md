# Collective Chess

Collective Chess is Reddit for chess positions — a community-built map where any position can be explored move by move. People submit candidate moves, explain the ideas behind them, and debate continuations. Over time, the best reasoning rises, and each explored position becomes a living reference for how to think, not just what to play.

## Monorepo

This repository will be organized as a monorepo.

- Backend: Laravel
- Frontend: Next.js
